    public toCents(amount: String): Number source

    Convert a currency amount into a 'cents' value

    !! Note: expects amount to be an amount divisible by 100, such as dollars or euros. If using a non-divisible amount such as Yen then do not use this function and use toIndivisible

    Params:

    NameTypeAttributeDescription
    amount String

    string representation of value

    Return:

    Number

    Example:

    toCents('$12.04') // => 1204
toCents(1000) // => 100000

    public toIndivisible(amount: String): Number source

    Remove any currency formatting and return as an Integer. This is specifically for dealing with non-divisible currencies such as Yen which cannot be divided into 'cents'. Also, this will round up (see example)

    Params:

    NameTypeAttributeDescription
    amount String
    • optional

    Return:

    Number

    Example:

    toIndivisible('-¥12,300') // => 12300
toIndivisible('-¥12,300.55') // => 12301
